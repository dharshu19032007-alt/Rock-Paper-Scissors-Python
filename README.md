# Rock-Paper-Scissors-Python
A simple Python Rock-Paper-Scissors game using basic conditions and random module.
The program then decides the winner based on the classic rules.

Features
User vs Computer gameplay
Randomized computer choices
Input validation
Clear win/lose/tie messages
Beginner-friendly, simple logic

Working
The user enters Rock, Paper, or Scissors
The computer randomly picks one option
Both choices are displayed
The winner is decided using the standard rules:
Rock beats Scissors
Scissors beats Paper
Paper beats Rock
